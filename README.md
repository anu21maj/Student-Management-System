# Student Management System (Java)

A simple console-based Java application designed to manage student records efficiently. This project demonstrates the use of core Java concepts and Object-Oriented Programming (OOP) principles.

---

## Features

- Add new student records  
- View all students  
- Search student by ID  
- Menu-driven user interface  

---

## Technologies Used

- Java  
- Object-Oriented Programming (OOP)  
- ArrayList (for dynamic data storage)  

---

## Concepts Implemented

- Classes and Objects  
- Encapsulation  
- Control Structures (loops, switch-case)  
- Collections (ArrayList)  

---

## How to Run

1. Clone the repository or download the file  
2. Open terminal in the project folder  
3. Compile the program:

```bash
javac Main.java
